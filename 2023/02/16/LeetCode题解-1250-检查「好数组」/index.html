🎯1250. 检查「好数组」 - 力扣（LeetCode）

📖题意

给定一个正整数数组 nums，你需要从中选出一个子集，然后给子集中的每一个数都乘以一个任意数（每个数可以乘的数可以不同，然后求出它们的和，如果和的结果为 1，则认为原数组是一个 好数组，如果是则返回 true 否则返回 false。

💡题解

涉及知识点：

  • 裴蜀定理（裴蜀定理 - 维基百科 ）
  • 最大公约数

由”n 个整数间的裴蜀定理”可知若这个数组中所有数的最大公约数为 1，则这个数组是一个好数组
