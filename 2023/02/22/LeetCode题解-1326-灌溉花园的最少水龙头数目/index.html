1326. 灌溉花园的最少水龙头数目 - 力扣（LeetCode）

题意

在一个一维坐标上有 n + 1 个点，从 0 点开始，到 n 点结束。在这 n + 1 个点的位置上都有一个"水龙头"，第 i 个点上的水龙头的覆盖范围为 [i - ranges[i], i + ranges[i]] 。现在需要求出覆盖着 n + 1 个点最少需要几个水龙头。

题解

可以将每个水龙头的覆盖范围理解为一个"线段"，然后求覆盖 [0, n + 1] 这个区间最少需要几条线段。其中需要注意，[1, 1]，[2, 2]这两条线段"不相交"，也就是说如果这个水龙头的 ranges[i] 为 0 那么啥也不是，自己都浇不了！

假设一个线段的表示为 M，其覆盖范围为[l, r] ，先来考虑简单的相邻两个线段覆盖区间的情况：

1、两个线段不相交

对于这种情况线段 M2 一定不会被选，因为两个线段间存在未被覆盖的区间，若想着段未被覆盖的区间被覆盖一定是在线段 M2之后存在覆盖区间小于 M1.R 的线段，此时线段 M2 会被这个线段完全覆盖(参考情况 3)，那么选 M2 就是多余的

2、两个线段相交重叠

若想区间被完全覆盖，那么相邻两个线段之间的关系一定都是这样的最终才是最优的
