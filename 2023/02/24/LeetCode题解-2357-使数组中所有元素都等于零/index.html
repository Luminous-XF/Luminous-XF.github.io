🎯2357. 使数组中所有元素都等于零 - 力扣（LeetCode）

📖题意

有一个非负整数数组 nums，可以进行如下两个操作

  • 选一个正整数 x，x 是 nums 中最小的正整数
  • 将 nums 中所有的正整数减去 x

求将 nums 中所有元素变为 0 需要最少的操作数

💡题解

考虑几个简单的情况

[1], 需要 1 次操作

  • x = 1, [1] => [0]

[1, 2], 需要 2 次操作

  • x = 1, [1, 2] => [0, 1]

  • x = 1, [0, 1] => [0, 0]

[1, 2, 3]，需要 3 次操作

  • x = 1, [1, 2, 3] => [0, 1, 2]
  • x = 1, [0, 1, 2] => [0, 0, 1]
  • x = 1, [0, 0, 1] => [0, 0, 0]

可以发现其实每次操作都是将最小的那些数变为 0，所以统计一下 nums 中除 0 外有多少种数
