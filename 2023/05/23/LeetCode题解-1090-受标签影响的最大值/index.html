🎯1090. 受标签影响的最大值 - 力扣（LeetCode）

📖题意

有 n 个元素，每个元素 i 有对应的值 value 和标签 label。现在需要从这 n 个元素中选择一个不超过 numWant 个元素子集，要求子集中不存在超过 useLimit 个元素的标签 label 相同且使子集，求该子集中所有元素值的和最大可以是多少。

💡题解

想要让取到的值的和最大，那么容易想到在符合 useLimit 和 label 限制的情况下尽可能取 value 大的元素，那么就对所有元素按照 value 的大小降序排序，然后从大到小取元素并在取的过程中用 map 记录某个 label 已取元素的个数，如果某个 label 取的个数超过了 useLimit 则该元素无法取到则直接跳过。
